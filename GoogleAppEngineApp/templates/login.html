What is Oboe?

Oboe is a free flashcard app designed for Japanese language sutdents, that utilizes spaced repetition using the sm2 algorithm. "Oboe" (覚え) is the Japanese word for memory, also the stem of the word "oboeru" (覚える) which means to memorize or to learn by heart.

How do I use Oboe?

To start using oboe, simply create a deck by clicking on the Add Deck button on the navbar at the top of the page. Once done, add cards to that deck by selecting the deck you wish to add the following card to, and fill out the input boxes. Home will direct you to your list of decks, to start studying a deck, click on that deck. You can rename, delete or change the amount of repetitions for each day by clicking the dropdown button called Actions.

How does it work?

Oboe is an interval in minutes based SRS application. When a user clicks on a button, it changes the interval of that card based on a multiplier. Below displays the result of each button press on the study page:

AGAIN: changes the interval of which the user will next see that card to 1 minute.

HARD: changes the interval of which the user will next see that card to interval * 1.2. If the current card interval is 10(10 minutes until next revision), then pressing HARD will result in the next revision for that card to 12 minutes

GOOD: changes the interval of which the user will next see that card to interval * 1.6. If the current card interval is 10(10 minutes until next revision), then pressing GOOD will result in the next revision for that card to 16 minutes

EASY: changes the interval of which the user will next see that card to interval * 1.9. If the current card interval is 10(10 minutes until next revision), then pressing EASY will result in the next revision for that card to 19 minutes
